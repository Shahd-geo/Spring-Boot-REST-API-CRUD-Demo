# Spring Boot REST API CRUD Demo

A simple CRUD REST API built using Spring Boot, MySQL, Lombok, and Postman.

This project was initially created by following a tutorial and then extended through additional practice and custom modifications.

## Features

* Create records
* Read records
* Update records
* Delete records
* MySQL database integration
* RESTful API design
* Postman testing

## Technologies Used

* Java
* Spring Boot
* MySQL
* Lombok
* Maven
* Postman

## Learning Outcomes

* Building REST APIs with Spring Boot
* Working with MySQL databases
* Using Lombok annotations
* Handling CRUD operations
* Testing APIs with Postman
* Understanding controller-service-repository architecture
